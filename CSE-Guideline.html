🎓 BTech CSE — 4 Year Roadmap

📅 Year 1 — ভিত্তি গড়ো (Foundation)

Semester 1

  • C Programming — loops, functions, arrays, pointers
  • Basic Math — Discrete Mathematics, Logic Gates
  • Computer Fundamentals — OS basics, how internet works
  • Typing speed বাড়াও (40+ WPM)

Semester 2

  • Python basics — C জানলে Python সহজ লাগবে
  • Git & GitHub — account খোলো, daily use করো
  • OOP concepts — Java বা C++ দিয়ে শেখো
  • ছোট project বানাও (calculator, quiz app, to-do list)
Year 1 শেষে Target:
  • একটা language ভালোভাবে জানবে
  • GitHub এ 2-3টা project থাকবে
  • কোন field ভালো লাগে সেটা বুঝে যাবে
📅 Year 2 — দক্ষতা বাড়াও (Skill Building)

Semester 3

  • DSA (Data Structures & Algorithms) — সবচেয়ে গুরুত্বপূর্ণ বছর
  • Array, LinkedList, Stack, Queue, Tree, Graph
  • Database — MySQL, basic SQL queries
  • Linux/Terminal — commands শেখো, developer দের লাগবেই
  • LeetCode এ Easy problems শুরু করো

Semester 4

  • DSA continue — Sorting, Searching, Recursion, Dynamic Programming
  • একটা Domain বেছে নাও:
    • 🌐 Web Dev → HTML, CSS, JavaScript, React
    • 📱 App Dev → Flutter বা Android (Kotlin)
    • 🤖 AI/ML → NumPy, Pandas, Scikit-learn
    • ☁️ Cloud/DevOps → AWS basics, Docker
  • LeetCode Medium problems শুরু করো
Year 2 শেষে Target:
  • DSA এর core topics শেষ
  • একটা domain এ কাজ শুরু হয়েছে
  • GitHub এ active থাকো
  • LinkedIn profile বানাও
📅 Year 3 — Real World তৈরি হও (Industry Ready)

Semester 5

  • Chosen domain এ deep dive করো
    • Web: Node.js/Express, REST API, MongoDB
    • AI/ML: TensorFlow/PyTorch, projects
    • App: Full app deploy করো Play Store এ
  • System Design basics শেখো
  • Open Source Contribution শুরু করো (GitHub)
  • Internship খোঁজা শুরু করো — LinkedIn, Internshala, Wellfound

Semester 6

  • Internship — যেকোনো একটা পেলেই করো, stipend কম হলেও
  • Portfolio website বানাও
  • 2-3টা বড় project বানাও resume এর জন্য
  • LeetCode Hard problems চেষ্টা করো
  • FAANG/Product company preparation শুরু করো (যদি সেটা লক্ষ্য থাকে)
Year 3 শেষে Target:
  • কমপক্ষে 1টা internship করা
  • 3-4টা solid project
  • Resume ready
  • 100+ LeetCode solved
📅 Year 4 — Placement & Future

Semester 7

  • Campus Placement Preparation:
    • DSA revision করো
    • Mock interviews দাও (Pramp, Interviewbit)
    • HR round practice করো
  • যদি Higher Study চাও (MS/MTech):
    • GATE preparation শুরু করো
    • GRE (বিদেশের জন্য)
    • Research papers পড়া শুরু করো
  • Final Year Project শুরু করো — ভালো topic বেছে নাও

Semester 8

  • Final Year Project শেষ করো, deploy করো
  • Placement/Offers handle করো
  • যদি startup করতে চাও — idea validate করো
  • Certificates নাও: AWS, Google Cloud, Meta Developer ইত্যাদি
Year 4 শেষে Target:
  • Job offer বা MS admission
  • Strong portfolio & GitHub
  • Clear career direction
